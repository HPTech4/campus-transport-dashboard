# Student Wallet Dashboard

## Overview
Frontend-only Student Wallet Dashboard for a campus cashless transport system using mock data.

## Stack
- React + Vite
- React Router

## Features
- Wallet Home
- Top-Up UI (mock virtual account)
- Ride History (mock data)
- Mobile-friendly layout

## Run Locally
```bash
npm install
npm run dev
```

## Notes
- Frontend only
- No backend or payment integration
- Uses mock data
